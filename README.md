Propiedades Termodinamicas del Agua
===========================================

Proyecto HTML5 + Javascript + SQLite para calcular las propiedades termodinamicas del agua

## Tecnologías

* HTML5
* Javascript
* JQuery
* IUI-JS
* WebSQL (SQLite deprecated HTML5 - Se Buscara Mudarlo a WebStorage)

## Compatibilidad
* Google Chrome
* Safari
* Safari Mobile
* Google Chrome Android 4.x
* Algunos navegadores de Android 2.x
